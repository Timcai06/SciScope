\section{后续补数与报告完善路线}

\begin{reportbox}{下一阶段目标}
当前已经跨过约 5 万篇公开论文数据资产基线。下一阶段重点不是继续堆数量, 而是把数据清洗、近五年趋势洞察、作者社区发现和智能体证据链做深, 让 PDF 中每个结论都可由代码复现。
\end{reportbox}

\subsection{优先级}

\begin{enumerate}
  \item 固化 \texttt{papers\_corpus\_50k.json} 为智能体检索、推荐、趋势分析的统一语料入口。
  \item 深化 DOI/title/year 去重与字段补全 pipeline, 对作者、机构、年份异常进行质量审计。
  \item 在当前图表资产之上补充领域分布、突增关键词、主题模型和社区发现结果。
  \item 将作者合作网络从静态核心子图升级为前端交互式全量图谱查询。
  \item 将报告资产接入前端 Dashboard、Trends、Graph 与 Ask, 形成产品化演示闭环。
\end{enumerate}
