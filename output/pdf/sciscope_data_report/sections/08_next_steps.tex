\section{数据层补强与下一步路线}

\begin{reportbox}{今晚优先级}
当前已经跨过约 5 万篇公开论文数据资产基线。下一阶段重点不是继续堆总量, 而是修正年份偏置、提升正文证据覆盖、补齐 RAG/Agent 所需字段, 让后续问答、推荐、趋势和图谱模块建立在更可靠的数据底座上。
\end{reportbox}

\subsection{执行优先级}

\begin{enumerate}
  \item 跑 \texttt{make data-layer-audit}, 固化年份、正文、字段覆盖缺口。
  \item 建立 2022--2025 年份均衡回补批次, 暂停继续追加 2026-heavy 数据。
  \item 扩展正文与证据片段来源, 先把 full-text/evidence records 推向 15k。
  \item 升级 canonical schema, 显式落 \texttt{doi}, \texttt{venue}, \texttt{url}, \texttt{citation\_count}, \texttt{text\_source}, \texttt{chunk\_count}。
  \item 在稳定数据层上构建 RAG chunks、FTS/vector index、趋势模型、推荐特征和图谱资产。
\end{enumerate}
