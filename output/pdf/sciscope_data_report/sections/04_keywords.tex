\section{关键词演化与热点趋势}

\begin{reportbox}{分析目标}
关键词演化是本赛题最容易形成“洞见感”的部分。最终报告不只列 Top-K 关键词, 而要展示关键词如何出现、升温、迁移、分化, 并说明这些变化如何驱动智能体的热点预测和论文推荐。
\end{reportbox}

\subsection{关键词处理策略}

\begin{enumerate}
  \item 统一大小写、连字符、复数和同义词, 例如 \texttt{retrieval-augmented generation} 与 \texttt{retrieval augmented generation} 归并。
  \item 从标题、摘要、数据库关键词、OpenAlex concepts、arXiv categories 中构建候选词。
  \item 使用领域词表过滤高频泛词, 例如 \texttt{study}, \texttt{method}, \texttt{model}。
  \item 为每个关键词生成年度频次、增长率、突增分数和代表论文。
\end{enumerate}

\subsection{热点预测口径}

趋势预测不应直接声称“未来一定热门”。更稳妥的表达是:

\begin{plainbox}
基于近年频次增长、跨领域扩散、代表论文质量、作者合作密度和引用信号, 将候选方向分成“强上升”“稳定增长”“早期萌芽”“可能噪声”四类。当前 PDF 已给出关键词升温候选榜, 后续系统侧将其接入趋势 API 与自然语言解释。
\end{plainbox}
