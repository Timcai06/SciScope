\section{科研智能体系统设计}

\begin{designbox}{产品定位}
科研智能体不是一个普通聊天框, 而是一个围绕科技文献构建的研究工作台。它要把数据分析报告中的静态洞察转化为可交互能力: 用户能继续追问、筛选证据、跳转图谱、获得推荐, 并理解答案来自哪些文献。
\end{designbox}

\subsection{可信答案原则}

\begin{itemize}
  \item 答案必须绑定证据卡, 证据卡必须能追溯到论文标题、年份、作者和片段。
  \item 趋势判断必须同时给出数据依据和不确定性, 避免把短期波动说成确定结论。
  \item 推荐结果必须解释“为什么推荐”, 至少包含主题相似、关键词重叠、作者关系或时间新鲜度之一。
  \item 图谱查询必须区分真实合作边、关键词共现边和模型推断关系。
\end{itemize}
